# AUTO BUILDER Frontend Ceiling Workbook

This package installs the frontend ceiling workbook as the operator ledger for frontend planning and execution.

## Install path

`01_Builder_Docs/frontend-ceiling/AUTO_BUILDER_FRONTEND_CEILING_WORKBOOK.xlsx`

## What it governs

- frontend planning
- UI implementation
- component rules
- responsive behavior
- accessibility
- performance
- forms
- frontend data contracts
- Vercel/Next.js runtime planning
- Supabase frontend contracts
- validation
- receipts
- release gates

## Source-truth relationship

Use this workbook together with:

`01_Builder_Docs/highest-repo-ceiling-all-in-one/AUTO_BUILDER_HIGHEST_REPO_CEILING_ALL_IN_ONE_WORKBOOK.xlsx`

The all-in-one workbook governs the broader ceiling and release-gate layer. This frontend workbook governs frontend-specific execution. Neither should overwrite the other.

## Production status

This workbook does not prove production readiness. Production remains locked until external live receipts exist.
