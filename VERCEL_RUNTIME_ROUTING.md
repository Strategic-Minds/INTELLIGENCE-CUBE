# Vercel Runtime Routing

Use Vercel only when a workbook calls for runtime behavior.

## Routing

- Vercel preview: safe app preview before production.
- Vercel Cron: scheduled runtime heartbeat or validator routes.
- Vercel Workflows: durable multi-step processes.
- Vercel Sandbox: isolated execution/testing where needed.
- Vercel AI Gateway: model routing, budget controls, fallback planning, and observability.

## Locks

Production deploys require explicit operator approval. Environment values are referenced by name only and managed outside the repo.
